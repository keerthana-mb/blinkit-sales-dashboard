# 📊 Task 2 - Data Visualization and Storytelling

## Blinkit Sales Dashboard

### 📌 Objective
The objective of this project is to create an interactive Power BI dashboard that transforms raw Blinkit sales data into meaningful business insights through effective data visualization and storytelling.

---

## 🛠️ Tools Used

- Microsoft Power BI Desktop
- Microsoft Excel (CSV Files)
- GitHub

---

## 📂 Dataset

The dashboard was created using a relational Blinkit sales dataset consisting of four tables:

- blinkit_customers.csv
- blinkit_orders.csv
- blinkit_order_items.csv
- blinkit_products.csv

Relationships were established between these tables to enable accurate analysis and reporting.

---

## 📈 Dashboard Features

The dashboard includes the following key performance indicators (KPIs):

- Total Sales
- Total Orders
- Total Customers
- Total Products
- Average Order Value
- Total Quantity Sold

### Visualizations

- Sales Trend Over Time
- Top 10 Products by Sales
- Orders by Delivery Status
- Sales by Category
- Customers by Segment
- Sales & Quantity by Category

### Interactive Filters

- Product Category
- Payment Method
- Customer Segment
- Delivery Status

---

## 📊 Business Insights

- Analyzed overall sales performance and order trends.
- Identified the top-selling products based on total sales.
- Compared sales across different product categories.
- Evaluated customer distribution by segment.
- Analyzed delivery performance using delivery status.
- Compared sales and quantity sold across product categories.
- Built an interactive dashboard with filters for better data exploration.
